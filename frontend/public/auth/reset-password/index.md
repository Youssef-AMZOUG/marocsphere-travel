# Reset Password - MarocSphere

> Set a new password for your MarocSphere account.

**Website:** [https://marocsphere.com/auth/reset-password](https://marocsphere.com/auth/reset-password)

---

## Set Your New Password

Use this page to set a new password after receiving a reset link via email. Your reset token is valid for 1 hour.

### Requirements:
- Enter your new password
- Confirm the new password
- Click "Reset Password"

After resetting, you'll be redirected to [Sign In](/auth/signin).
